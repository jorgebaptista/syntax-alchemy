# Strings

String literals, concatenation, and comparisons.

## Literals

Use double quotes for string literals.

```
print "hello"
```
Output: `hello`

## Concatenation

Use `+` to concatenate two strings.

```
print "foo" + "bar"
```
Output: `foobar`

## Comparisons

Strings support `==`, `!=`, `<`, `<=`, `>`, `>=`.

```
print "a" < "b"
```
Output: `True`

## Printing

`print` outputs the string value without quotes.

## Type Errors

Only string + string is supported. Other combinations are type errors.
